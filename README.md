# BoorIOS

**BoorIOS** (formerly BoriOuS) is a modern, feature-rich Booru image viewer for iOS and iPadOS. Built with SwiftUI and SwiftData, it offers a premium browsing experience for your favorite image boards.

## Features

- **Multi-Source Support**: 
  - **Danbooru** (SFW)
  - **Safebooru** (SFW)
  - **Moebooru** (Konachan, Yande.re)
  - **Gelbooru** Family: e621, e926, Rule34, Gelbooru (customizable)
- **Modern UI**: Full-screen immersive viewer, masonry grid layout, and iPad sidebar support.
- **Offline Favorites**: Save your favorite posts to the device database.
- **Tag Search**: Powerful tag search with blacklist support.
- **Download & Share**: Save high-quality images to your Photo Library or share them directly.
- **Customizable**: Light/Dark mode, grid columns, content filtering (SFW/NSFW), and more.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/elouann/BoorIOS.git
   ```
2. Open `BoriOuS.xcodeproj` in Xcode 15+.
3. Select your development team in Project Settings -> Signing & Capabilities.
4. Build and Run on your device or simulator.

*Note: The project requires iOS 17.0+ for modern SwiftUI features.*
